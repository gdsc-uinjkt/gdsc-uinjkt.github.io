What is Google Developer Student Clubs?

Google Developer Student Clubs (GDSC) is a program from Google Developers for students designed to help improve students' skills and knowledge in developing skills for mobile applications, web, machine learning, and other technologies. This program is open to active students from all majors at UIN Syarif Hidayatullah Jakarta.
